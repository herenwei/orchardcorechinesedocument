    Start Bootstrap - Agency

    [Start Bootstrap - Agency]（https://startbootstrap.com/template-overviews/agency/）

    Agency is a one page agency portfolio theme for Bootstrap created by Start Bootstrap. This theme features several content sections, a responsive portfolio grid with hover effects, full page portfolio item modals, a responsive timeline, and a working PHP contact form.

    [Agency]（https://startbootstrap.com/template-overviews/agency/）是由[Start Bootstrap]创建的[Bootstrap]（http://getbootstrap.com/）的单页代理商组合主题（http：/ /startbootstrap.com/）。此主题包含几个内容部分，具有悬停效果的响应式投资组合网格，完整页面投资组合项目模式，响应时间轴和工作的PHP联系表单。

    Usage

    用法

    Basic Usage

    基本用法

    After downloading, simply edit the HTML and CSS files included with the template in your favorite text editor to make changes. These are the only files you need to worry about, you can ignore everything else! To preview the changes you make to the code, you can open the index.html file in your web browser.

    下载后，只需在您喜欢的文本编辑器中编辑模板中包含的HTML和CSS文件即可进行更改。这些是你需要担心的唯一文件，你可以忽略其他一切！要预览对代码所做的更改，可以在Web浏览器中打开index.html文件。

    Advanced Usage

    高级用法

    After installation, run npm install and then run gulp dev which will open up a preview of the template in your default browser, watch for changes to core template files, and live reload the browser when changes are saved. You can view the gulpfile.js to see which tasks are included with the dev environment.

    安装完成后，运行npm install，然后运行gulp dev，它将在默认浏览器中打开模板预览，监视核心模板文件的更改，并在保存更改时实时重新加载浏览器。您可以查看gulpfile.js以查看dev环境中包含哪些任务。

    Gulp Tasks

    吞咽任务

    • gulp the default task that builds everything

    • gulp是构建所有内容的默认任务

    • gulp dev browserSync opens the project in your default browser and live reloads when changes are made

    • gulp dev browserSync在默认浏览器中打开项目，并在进行更改时进行实时重新加载

    • gulp sass compiles SCSS files into CSS

    • gulp sass将SCSS文件编译成CSS

    • gulp minify-css minifies the compiled CSS file

    • gulp minify-css缩小编译后的CSS文件

    • gulp minify-js minifies the themes JS file

    • gulp minify-js缩小主题JS文件

    • gulp copy copies dependencies from node_modules to the vendor directory

    • gulp copy将从node_modules的依赖项复制到vendor目录
